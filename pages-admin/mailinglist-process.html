Newsletter Process

Anyone can submit a section to be sent to the mailing list, either in a specific email or in the next newsletter

Use of the mailing list

All sections should aim to be potentially relavant for everyone on the mailing list.

If event details are in the submitted draft, it must have followed the CodeWith Event Process.
