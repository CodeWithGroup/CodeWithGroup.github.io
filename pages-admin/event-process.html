Event Process

When do I need to follow this process?

This process should be followed if you need any support from the CodeWith team when setting up orrunning your event this includes:

  • Mentors
  • Ticket management (Eventbrite)
  • Slide deck help
  • Use of the CodeWith branding
  • Advertising on CodeWith channels (Mailchimp, Facebook, Twitter, Slack)
