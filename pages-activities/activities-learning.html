Learning

Codecademy Easy

Code Academy gives you specific problems to solve and, as you solve each one, the difficulty level increases. You can learn about building websites, developing programs (in languages like Python) and get an introduction to Data Analysis.

Code Combat Easy

Code Combat was designed for use in schools but is a great place to start if you're looking to jump straight in. There are about 10 lessons where you get used to the basics but before long you're designing and implementing the AI for a game character.

CSS Diner Easy

This is a simple game that introduces you to different CSS selectors in an interactive and fun way, currently it has 30 stages which cover most of the main and recent CSS selectors you should be aware of as a beginner.

CSS Tricks Easy

CSS Tricks is an excellent website with a large amount of information on CSS, including code snippets, guides, and even job postings!

Erase All Kittens Easy

E.A.K. is a revolutionary, online game that provides kids aged 8-14 with knowledge of both computational thinking and professional coding languages, to effectively prepare them for 21st Century degrees and careers.

FlexBox Froggy Easy

Flexbox Froggy is a game for learning CSS flexbox.
It takes inspiration from the classic arcade game Frogger, and web literacy games like the brilliant CSS Diner and Erase All Kittens, where you learn about CSS selectors and HTML markup respectively.

FreeCodeCamp Easy

FreeCodeCamp is a popular online learning tool for beginners to get to grips with new langauges and concepts. It offers training in different aspects of software and has challenges of varying difficulty.

git - the simple guide Medium

Just a simple guide for getting started with git, no deep or confusing stuff. Good as an introduction to the terminology before starting to read other guides which go into more detail.

Micro:Bit Easy

The BBC micro:bit is a pocket-sized computer that introduces you to how software and hardware work together. It has an LED light display, buttons, sensors and many input/output features that, when programmed, let it interact with you and your world.

The Modern JS Tutorial Easy

Has its own built in search function, using which you can search every aspect of javascript for helpful guides and examples.

Udemy Easy

Udemy is an online platform offering video tutorial courses led by real-world professionals. There is a wide range of courses, some free, some paid, covering all sorts of things (not just development).

Untrusted Javascript Medium

The game presents you with a roguelike-like playing environment and a console window with the JavaScript code generating each level. As loaded, each level is unbeatable, and most of the JavaScript is blocked from editing. The challenge is to open a path to the next level using only the limited tools left open to you.

W3 Schools SQL Easy

An easy to follow tutorial through SQL. Teaches you basic queries, terminology and commands to get you started on your SQL adventure. Also a small quiz at the end of each topic to review what you had learned.
