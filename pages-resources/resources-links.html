Useful Links

GitHub

Git is industry-leading version control software and is incredibly popular with professionals and hobbyists alike. CodeWith has a public GitHub repository where we encourage you to store any public projects you work on at CodeUp. We even run this web site direct from our GitHub repository.

Slack

CodeWith has a Slack workspace where you can chat, share files, ask questions and get help with problems in between Meetups. Follow the link to get an invitation.
